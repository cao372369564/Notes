================常用 查看文件命令================================ pwd 查看当前所在地址 /windows下面查看当前路径的文件 dir cd ~ 当前用户的主文件夹 cd -ll 查看当前目录都有哪些文件 cd -ls cd -al 显示隐藏文件 打开某个文件 vim a.txt --------------------------------分割线--------------------------------------- 文件名： bin 可执行的文件 etc 用户配置文件 home 用户文件夹(每多一个用户都会多一个文件夹) 只有 root 在根目录文件夹下，其他的用户都在home文件夹下 root lib linux下的函数库 Midia 存放一些可删除的设备，U盘，光盘 mnt 和midia功能差不多 opt 存放第三方软件的目录 sbin 和bin文件一样，但只允许系统管理员访问 srv 用户主动生产的数据 tmp 临时文件 proc 虚拟文件系统,不在硬盘上存储，它储存在内存上，开关机就没了 sys 虚拟内核系统，和系统内核相关的进程 -----------------------------分割线--------------------------------------- 创建文件 mkdir 一次只能创建一个，不能创建嵌套文件 mkdir -p 可以一次创建多个嵌套文件 删除文件 rmdir 只能删除空文件夹 rm -rf 删除有内容的文件夹（慎用） ------------------------------------------分割线---------------------------- 文件权限 第一个字符 ： 路径还是文件(D表示路径，-标识文件) 分三组,每组三个字符 第一组，当前所属用户的权限 第二组，当前所属组的权限 第三组，其他用户的权限 - - - r w x 4 2 1 数字表示权限值 有可读可写的权限就是4+2 也就是6 有可写可执行 就是2+1 就是3 可读可写可执行 就是4+2+2 是7 R 表示有权限读 w 表示有权限写 x 表示有权限执行 (例子： R-- 表示只能读 不能写不能执行 -w- 表示只能写 不能读不能执行 --x 表示只能执行 不能读不能写) 例子： 创建文件赋权限 mkdir -m 765 (-m可赋予权限) 加权限 chmod 777 -------------------------------------分割线-------------------------------- 系统自带命令 vi (意味进入vim编辑器) 打开记事本 vi a.txt (只是打开，并未被保存) 写入 先按 i 左下标显示--INSERT--就可以写东西了 先按ESC，然后打出冒号 成功后左下角变成冒号，说明可以输入命令 保存退出 : wq 然后回车 强制退出不保存 ：q! q加英文感叹号 然后回车 shift + $ 行头 shift + ^ 行尾 ctrl + f 上 ctrl + b 下 tail 查看最后10行 eg：tail notepad tail -100 查看最后100行 其他linux 记事本操作请百度搜索 -----------------------------------------------分割线---------------------
